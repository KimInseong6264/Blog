# Ⅱ. 변수

### 1.변수
: 문자열 or 숫자열을 저장할 수 있는 공간
- 변수를 지정을 해야만 변수에 값을 저장할 수 있음

### 2. 변수명 규칙
- 변수명에 공백, 특수문자 사용x
- 변수명에 예약 키워드는 사용x
- 변수명이 길다면 캐멀케이스로 여러 단어를 묶어서 하나로 표현

### 3. 변수의 자료형
> int 정수형 : 정수를 지정하는 자료형
char(캐릭터형) : 한 개의 문자만을 지정하는 자료형
string : 문자열을 지정하는 자료형
float : 짧은 실수형을 지정하는 자료형
double : 긴 실수형을 지정하는 자료형
bool : '참' 혹은 '거짓'을 지정하는 자료형
short : 짧은 정수를 지정하는 자료형
long : 긴 정수를 지정하는 자료형
