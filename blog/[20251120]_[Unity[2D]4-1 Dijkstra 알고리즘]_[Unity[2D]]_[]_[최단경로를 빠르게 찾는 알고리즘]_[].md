# Dijkstra 알고리즘

### 길 찾기 알고리즘
- F(최종 비용) = G(인접 노드 간의 거리)
1. 내 주변 레이더를 한 바퀴 돌린다.
2. 레이더에 걸리는 이동 가능 지점들의 각 이동비용을 openList에 정리
3. 비용이 가장 저렴한 경로로 이동하고 visitedList에 체크
   (비용은 방문기록이 없는 경로는 이미 이동한 비용까지 누적해서 기록)
4. 위의 내용을 반복
5. 도착지에 도착할 시에 이전 


#### 사용 List
1. openList : 방문 직전 
2. closedList
3. G
