# Ⅰ. 배열&foreach문

### 1. 배열
---

**※ 배열:** 동일한 타입의 여러 데이터 요소를 1개의 변수에 저장하고 관리하는 형식
- 서로 같은 자료형을 가져야 함
- Heap이라는 데이터 공간에서 값을 저장할 공간을 미리 할당하고 그 공간에 값을 저장
- 우리가 사용하는 배열은 그 공간의 값을 참조(어디에 값이 저장되어 있다는 것)만을 가리키고 있는 형식

> **입력형태**<br>
> string[] a(함수명)  = new string[n(빌릴 공간의 수)];<br>
  ▶ new : 공간을 빌려달라는 연산자<br>
  ▶ string 이외에 int , float 등을 선언가능

![](https://velog.velcdn.com/images/iissk/post/5057857f-008f-4617-8bd7-f66f9c42bd8b/image.png)
1) 배열 공간을 만들어 놓는다.
2) 공간을 몇 개로 쪼개 놓을지 지정한다.

※값 타입 : 값 자체가 저장되는 것
- 4바이트 공간 하나만 만들어짐
참조 타입 : 값이 아닌 위치정보를 저장하는 것
- 특정한 공간을 형성해놓고 주소만 기억하는 것
<br>
프로그램은 배열을 0부터 위치를 센다.


![](https://velog.velcdn.com/images/iissk/post/c33f7a63-3204-40a1-97d7-edf70f7a64b2/image.png)
▶ 단순하게 초기화도 가능하다.

![](https://velog.velcdn.com/images/iissk/post/30c18122-96eb-4aba-9699-80c7da10c8b2/image.png)
▶ 배열의 각 위치에 값을 지정해준다.

---

![](https://velog.velcdn.com/images/iissk/post/1cb2f842-34e9-4f82-8465-918547b579c9/image.png)

▶ foreach문을 통해 배열의 반복이 가능

**※ foreach :** 배열의 반복 출력은 되지만 읽기 전용으로 대입or연산등의 수정이 불가능
-  하위 배열에 대해서 컴퓨터가 자동으로 추적을 해주는 특징으로 거대한 스케일의 코딩에 유리함


### 3. 자료형 var
---

**※ var :** 변수 우항에 입력되는 값에 따라 자료형을 컴퓨터가 자동으로 지정해주는 자료형
- 한 번 자료형이 정해지면 그 이후로 변동은 어려움
- 코드의 길이가 길어지고 배열등으로 많은 자료를 담게 될 경우 여러 자료형이 복합적으로 작용하기에 에러를 줄일 수 있음
- *var을 남용할 경우 코드의 가독성이 떨어지고 협업 시에 소통의 문제가 발생함*

![](https://velog.velcdn.com/images/iissk/post/9fa7b5f5-83ba-4aa5-b904-2db90b5ee25c/image.png)


### 4. 문자열 = char형의 배열

![](https://velog.velcdn.com/images/iissk/post/02f85d8f-a62a-43b8-9d01-a33367a64b6d/image.png)

▶ 컴퓨터는 Ingrid는 5개의 자리(공간)에 알파벳 1개(char형)씩 배열 된 상태로 인식<br>
▶ name의 5번째 배열값인 "d"가 출력 되게 됨


### 5. 배열의 차원
---
**※차원:** 나뉘어진 배열을 다시 한 번 배열화하여 2개 이상의 배열값을 가지는 것
ex) xy 좌표평면 , 공간의 위치정보(3차원)


![](https://velog.velcdn.com/images/iissk/post/f51ba4dd-25d6-4efd-adef-b785045d7be8/image.png)
▶ 가장 바깥쪽 중첩(3번째 행) 수준은 가장 왼쪽 차원[3]에 해당하고 가장 안쪽 중첩(1번째 열) 수준은 가장 오른쪽 차원[1]에 해당합니다.<br>
▶ 바깥쪽부터 배열값을 세면 된다.


![](https://velog.velcdn.com/images/iissk/post/4225576f-e937-467e-adf0-5f2246d52c0f/image.png)
▶ 2차원 배열은 행렬을 표현한다.<br>
▶ 인덱서의 값이 각각 [행, 열] 을 의미한다.

<br> 쉽게 생각하면 책장에 동일한 시리즈의 책을 차곡차곡 집어 넣는 것처럼, **Heap 공간에 데이터 공간을 미리 할당하고 이후에는 이 공간에 비슷한 성질의 값들을 순서대로 넣는 것**을 배열이라고 한다.
