# Ⅰ. 구조체의 생성자

### 1. 생성자
---
**※생성자:** 클래스에서 객체를 생성할 때 매개변수만으로 빠르게 인스턴스를 만드는 메서드의 일종
- 단축키: **ctor**
- 모든 클래스에는 '인자값을 요구하지 않는 기본 생성자'가 있지만 인자값을 1개라도 요구하는 생성자를 만들 경우, 기본 생성자는 사라진다.
- 생성자는 new를 통해서 실행을 시킨다. (new 생성자)

#### 기본형식

```
public [클래스명] (원하는 매개변수)
{
    생성과 동시에 실행이 되었으면 하는 코드
}
```
