# Ⅱ. Random


**※Random:** 랜덤한 값을 지정하도록 콘솔 내에 저장된 자료형
- 참조타입으로 new 를 사용해서 초기화를 해줘야 한다.

![](https://velog.velcdn.com/images/iissk/post/9b68a3f0-ea11-409e-b5cf-eeebbb2a66a6/image.png)


**※랜덤값 생성**
![](https://velog.velcdn.com/images/iissk/post/dbe8a63b-c3cd-4570-9409-5953dd742ffc/image.png)

▶ **변수명.Next(a , b) :** a ≤ 랜덤수 ＜ b 에 해당하는 랜덤수를 생성
<br>
ex) 랜덤한 주사위값을 출력, 특정한 수를 랜덤으로 출력해야 할 때 사용
