1. 게임 매니저
2. 캐릭터
3. 에너미
4. 스킬
5. 상태변화
6. 전투
7. 스테이지


## 2. 클래스 세분화

**1) 게임 매니저**
- 싱글톤 게임매니저
    - 다른 매니저들을 참조해서 사용
    - 게임매니저만 유일성을 지키며 매니저들에게 지시만 내림
- UI 매니저
    - 게임 전체 UI 총괄
    - 타이틀화면, 게임오버 관리
- 사운드 매니저
    - BGM 매니저
    - EffectSound 매니저
- 스폰 매니저
    - 캐릭터 선택 및 에너미 생성 관리
    - 디스폰까지 함께 관리
- 스테이지 매니저
    - 스테이지 선택지 랜덤 생성
    - 화면에 스테이지 화면 출력
        - 스테이지 전투 시작
        - 스테이지 전투 종료
- 씬 전환 매니저

<br>

**2) 캐릭터**(목표: 총4명)
- 유닛 베이스(에너미와 공유)
    - 추상 클래스 제작
    - 스텟
        - 공격력(데미지만 관여)
        - 방어력
        - hp
    - TakeDamage
- Weapon
    - 리스트(스킬2개 장착)

<br>

**3) 에너미**(목표: 일반2명 + 보스1명)
- 유닛 베이스(캐릭터와 공유)
- Weapon
    - 보스는 패턴을 위해 추가 제작예정

<br>

**4) 스킬**
- 캐릭당 2개의 스킬(목표: 총8개)
    - 기본위력
    - 코인위력
    - 코인수
- 스킬별 효과(버프, 디버프)

<br>

**5) 상태변화**
- 스텟 변화 클래스
    - 추상 클래스
    - 공격력 변화
    - 방어력 변화
    - 기본위력 변화
    - 코인위력 변화
- 버프
- 디버프

<br>

**6) 전투**
- 스킬 출력 및 선택
    - 랜덤 슬롯 출력
- 코인토스
    - 랜덤 코인토스
- 합 진행
- 데미지 계산
- 태그(캐릭터, 에너미) 전멸 시 스테이지 종료

<br>

**7) 스테이지**
- 각 스테이지 랜덤 생성
    - 3번의 일반 맵 출력
    - 4번 보스방 출력
