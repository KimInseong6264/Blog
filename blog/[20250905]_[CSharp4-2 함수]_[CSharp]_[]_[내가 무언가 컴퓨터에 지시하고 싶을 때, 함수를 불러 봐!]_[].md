
# Ⅱ. 함수

### 1. 함수
---
**※ 함수 :** 입력에 따른 출력을 만들어내는 중간 처리 코드
- 동일한 코드를 여러번 반복 입력해야 하는 경우 함수를 정의해서 **반복적인 작업을 간소화 가능**
- 간소화를 통해 코드 전체 **가독성이 크게 증가**하며 협업시에도 타인에게 정보를 쉽게 이해시킬 수 있음
- 동일한 반복 작업을 1개의 함수로 정의함으로서 각 시행마다 발생할 수 있는 **에러발생률이 크게 감소함**

**※ 코드블록 :** 여러개의 코드의 묶음을 지칭하는 단어

▶ **인자값(매개변수):** 계산을 위해 필요한 준비물을 넣어주는 값


### 3-1. 인자값, 반환값이 없는 함수
---
**※void :** 반환값을 지정하지 않는 함수의 자료형
- 반환값이 없다는 것은 Main공간에서 void 함수를 사용하더라도 함수 자체가 값을 가지고 있지 않다는 것
- void 함수는 다른 변수에 담을 수 없음

![](https://velog.velcdn.com/images/iissk/post/c81ce757-41ec-485d-8e0e-b7115997deaa/image.png)
▶ 반환값과 인자값 모두 없이 함수 입력시 "좌회전" 3번만 출력하고 종료됨

### 3-2. 인자값O, 반환값X 함수
---
**※** void 함수 괄호 안에 인자값을 입력하는 형식

![](https://velog.velcdn.com/images/iissk/post/ae867418-6874-4475-b493-0d6bd3a7f155/image.png)

(void 함수임으로 반환값은 없음)
