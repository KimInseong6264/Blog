### 그랩(시뮬레이터는 오른쪽 + G )

![](https://velog.velcdn.com/images/iissk/post/25224178-50af-4624-94b8-b12cef445414/image.png)
- 셀렉트 모드에서 멀티플로 바꾸면 양 손을 써야만 물체를 들 수 있음
- 어테취 트랜스폼에 원하는 위치를 넣어서 그 위치를 잡도록 할 수 있음
- 인터렉터블 이벤트
	- 엑티베이트 : 검지를 눌렀을 때, 동작하는 메서드를 등록
    - 호버 : 가져다 대기만 하면 동작하는 메서드 등록
    - 셀렉트 : 잡자마자 발동하는 것
- 무브먼트 타입
	- 키네마틱 : 방망이처럼 다른 물체와 충돌 가능
    - 벨로시티 : 속도기반으로 동작하며 바닥을 뚫지 못하게 함
- 쓰로우 디테치 : 던지지 못하게 막는 기능(무브먼트 타입을 인스턴티로 해야 함)
- 트랙 포지션(로테이션) : 잡을 수는 있지만 움직이거나 회전을 하지 못하도록 막는 것
- 포스 그레비티 : 처음엔 중력의 영향을 안 받다가, 그랩을 하면 중력의 영향을 받게 함(나무열매 같은 거에 사용)
- 유즈 다이나믹 어테치 : 우리가 누르는 곳이 잡는 위치가 됨(평소보다 연산을 조금 더 하게 됨)
- 리테인 트렌스폼 패런트 : 기본적으로 그랩하면 패런트에서 빠지는데, 이 기능이 설정되면 패런트로 다시 돌아감
